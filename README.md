# 蘑菇碳的Flutter金融理财App

> 喜欢请Star，尽量别 Fork。


### Node 后台：
1. 安装`node.js`
2. cd 到项目`back-end`目录下
3. 控制台运行`npm i`
4. 控制台运行`node server.js`
5. 启动服务端成功
